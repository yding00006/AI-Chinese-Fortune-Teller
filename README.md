# 🏮 Chinese Fortune Telling Website (中国算命网站)

A web application powered by AI that delivers personalized fortune-telling readings rooted in traditional Chinese divination methods.

## ✨ Features

### Core Divination Methods
- **八字 (Bazi)** — Four Pillars of Destiny: Analyze life path based on birth date and time
- **紫微斗数 (Zi Wei Dou Shu)** — Purple Star Astrology: Star chart-based destiny reading
- **易经 (I Ching)** — Book of Changes: Wisdom through hexagram consultation
- **面相 (Face Reading)** — Physiognomy: Personality and fortune through facial features

### Application Features
- 🎨 Immersive mystical Chinese aesthetic (ink wash textures, gold-and-dark palette)
- 📝 User input forms (birth date/time picker, question input)
- 📊 Formatted reading results display
- 🌐 Multilingual support (English + Chinese) — *planned*
- 📜 Reading history — *planned*
- 🔮 Daily fortune feature — *planned*
- 🔗 Social sharing — *planned*

## 🛠️ Tech Stack

- **Frontend**: HTML5, CSS3, JavaScript (Vanilla)
- **Backend**: Node.js + Express *(planned)*
- **AI Engine**: Anthropic Claude API *(integration pending)*
- **Styling**: Custom CSS with CSS variables, ink wash / gold theme
- **Fonts**: Noto Serif SC (Chinese), Cinzel (English headings), Cormorant Garamond (body)

## 🚀 Getting Started

### Prerequisites
- A modern web browser (Chrome, Firefox, Safari, Edge)

### Installation

```bash
# Clone the repository
git clone https://github.com/yding00006/AI-Chinese-Fortune-Teller.git
cd AI-Chinese-Fortune-Teller

# Open directly in browser
open index.html

# Or serve locally
npx serve .
```
